Os comandos do chat sempre deve começar com uma das opções abaixo. Use a que mais gostar:

> !cc
> !cabron
> !cabroncoin
> !cabr0ncoin
> !cabronc0in
> !cabr0nc0in

Seguem as funcionalidades e os seus comandos possíveis.
Menor que e maior que < > indicam parâmetro obrigatório.
Colchetes [ ] indicam parâmetro opcional.
Valores contendo espaços devem ser contidos entre aspas duplas.

# Ajuda
  Exibe o link para acessar este arquivo de ajuda.

> !cc ? "[idioma]"
> !cc -h "[idioma]"
> !cc --help "[idioma]"

  [idioma] : Atualmente disponíveis "pt-BR" e "en-US", que podem ser referenciado apenas como "pt" e "en", respectivamente.

# Carteira
  Registrar a carteira na blockchain.
  Carteiras são criadas automaticamente quando necessário. Logo, este comando não tem tanta utilidade assim.

> !cc -wn
> !cc --wallet --new

# Perfil do usuário
  Registrar uma frase de status para o perfil do usuário na blockchain.

> !cc -p "[mensagem-de-status]"
> !cc --profile "[mensagem-de-status]"

  [mensagem-de-status] : Se não for informada será utiliza a frase padrão da blockchain.
